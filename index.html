Mikbotam
(Mikrotik bot telegram)

        Mikhbotam Merupakan aplikasi berbasis web perpaduan antara mikrotik dengan bot telegram Fungsi Utama manage hotspot mengunakan bot telegram, Mikbotam bukan lah hal yang baru namun mikbotam adalah trobosan baru untuk transaksi voucher,mikbotam berbeda dengan pada umumnya menjual voucher dengan hardcopy atau sejenis voucher print, mikhbotam menggunakan bot telegram dengan system saldo, menggunakan Mikrotik api class dengan bahasa pemograman PHP untuk berkomunikasi dengan mikrotik, Mikbotam bukan radius server

Meningkatkan jumlah pendapatan anda

Efesiensi bahan baku, Hemat kertas , Hemat Bahan bakar cukup praktis tinggal pencet tombol voucher keluar sendiri

Menurunkan beban resource MikroTik

Anda tidak perlu lagi membuat user sebanyak banyaknya untuk mendapatkan voucher cukup dengan menekan tombol pada bot otomatis user akan terbuat dengan sendirinya

Terus terhubung dengan jaringan anda dimana saja

Kontrol penjualan anda, jika anda sedang di luar kota anda masih bisa pantau secara Live dan akurat karena anda bisa mengaksesnya melalui bot dan web UI Tidak perlu khawatir kehabisan voucher

Perluas jaringan tanpa takut terlalu jauh

Anda tak perlu lagi memikirkan untuk mengirimkan voucher ke counter counter reseller warung-warung cukup dengan bot telegram mereka bisa menjual voucher anda, Silahkan duduk manis sambil ngopi menghitung keuntungan anda
